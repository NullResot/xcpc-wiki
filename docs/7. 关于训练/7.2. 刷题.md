# 7.2. 刷题

切记不要听信某些平台说只看xxx课就可以了，所谓训练就是要大量的刷题，无论什么网站只要有适合自己的好题就要去做。

codeforces的题目偏向于思维，atcoder的题目偏向于典题，我个人是觉得刚入门的话只需要做到大量刷题即可，最好的话是去刷高于自己当前最大rating + 300难度的题目。

非常推荐使用cftracker和atcoder problems这两个东西来做题。
